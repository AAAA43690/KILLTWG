# *KILLTWG*
## 為什麼會有這個東西?
因為明道給學生用的的電腦子系統管理做的太爛，但反應過後沒動作，所以用行動來表達不滿。

## 這個檔案可以幹嘛?
如果上課老師一直切畫面，導致你不高興，打開這個檔案，一次解決問題。

## 怎麼打開?
直接打開就可以了。

    1.打開 KILLTWG 之後會看到一個 "您是否要允許此 App 變更您的裝置" 請點擊 "是"

    2.接下來會有一個紅色視窗，它會問一個問題，如果要執行，請打上 "yes" ， 完成 !
    
    3.恭喜，你的電腦將不會再受到老師控制

## 遇到問題?
Q : 開啟檔案時遇到錯誤了，怎麼辦?

A : 試試看用管理員身分執行，作法如下

    1.對著 KILLTWG 點滑鼠右鍵，找到一個 "以系統管理員身分執行"
    2.按下去就好了
